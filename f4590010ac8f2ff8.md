# Vanguard Navigator

Vanguard Navigator 是一个面向互联网技术从业者与数字内容研究人员的轻量级外链资源聚合与导航系统。本项目不提供具体内容托管，而是以严格筛选的优质外部链接为索引，帮助用户快速定位高价值的技术文档、影视学术资料、运动科学资源及数字娱乐素材。项目定位为“技术型外链收藏夹即服务”，适用于个人研究者、小型团队知识管理以及自动化信息采集流程中的种子源配置。

## 功能概览

- **多源链接分类管理**：支持按内容领域（技术、影视、体育、娱乐）对原始外链进行标签化分类，便于用户按需检索。
- **链接存活状态检测**：内置轻量级 HTTP 头探测模块，可周期性检查收录链接的可达性，并在管理面板中标记异常状态。
- **自定义导入与导出**：允许用户通过 JSON 或 CSV 格式批量导入外部链接列表，并支持将当前索引库导出为静态 HTML 书签文件。
- **只读镜像缓存机制**：对频繁访问的链接内容生成文本级快照（仅限公开文本信息），避免源站临时宕机导致信息丢失。
- **RESTful API 查询接口**：提供基于关键词、分类标签和域名过滤的 JSON 格式查询接口，方便嵌入其他自动化工具或脚本。
- **访问热度统计**：记录各链接在系统内的点击次数与最后访问时间，辅助用户识别高频核心资源。
- **黑暗模式与阅读视图**：前端展示界面支持明暗主题切换，并针对文章类链接提供去噪阅读模式。

## 应用场景

- **技术文档快速入口**：开发人员可将项目作为个人或团队的技术书签中心，集中存放各类 API 文档、开源库主页及技术博客入口，配合关键词检索提升查阅效率。
- **影视素材索引管理**：影视剪辑师或内容研究者可利用本系统分类存储不同题材的影视素材网站、预告片源站或学术影评库，实现跨站资源统一调度。
- **运动科学数据采集**：体育科学专业的学生或教练可整理运动生物力学、训练方法及赛事录像相关的网络资源，通过本系统建立结构化学习路径。
- **数字娱乐资源聚合**：游戏开发者或娱乐产业分析师可汇集行业报告、游戏设计文章及独立游戏展示页面，形成垂直领域的外链知识库。
- **自动化采集种子配置**：数据工程师可将本系统导出的链接列表直接用作爬虫任务的初始 URL 种子集，配合 API 实现动态更新。

## 快速开始

以下步骤帮助您在本地环境中快速启动 Vanguard Navigator 实例。

```bash
# 克隆项目仓库至本地
git clone https://github.com/vanguard-navigator/vanguard-navigator.git

# 进入项目根目录并安装依赖（使用 npm）
cd vanguard-navigator
npm install

# 使用示例配置启动开发服务器
npm run start:dev
```

启动成功后，访问控制台输出的本地地址（默认为 http://localhost:3000）即可进入系统。首次启动会自动生成示例链接数据。

## 常见问题

**问：系统对收录的链接内容是否有版权审查义务？**

答：Vanguard Navigator 仅作为技术性外链索引工具，不存储、修改或分发任何第三方内容。系统生成的文本快照仅包含公开的 HTML 标题及元描述字段，且仅供个人研究使用。用户应遵守目标网站的服务条款，本项目不承担因链接内容引发的任何版权或法律纠纷责任。

**问：如何更新已收录链接的元数据（如标题或分类）？**

答：您可以通过前端管理面板的编辑功能手动修改，也可调用 API 的 `PUT /api/links/{id}` 接口进行更新。若希望自动刷新，可在系统配置中开启“周期性元数据重抓”选项，系统将每隔 72 小时重新提取链接标题及描述。

**问：系统支持 PostgreSQL 或其他关系型数据库吗？**

答：当前正式版本仅内置 SQLite 支持，以降低部署门槛。若需生产环境使用 PostgreSQL，可参考 `/docs/ops-guide.md` 中的扩展配置章节，通过修改数据库适配器连接字符串实现，但该模式尚未经过全面测试，建议谨慎评估。
