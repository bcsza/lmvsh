# LinkMatrix

LinkMatrix 是一个面向技术社区与内容创作者的轻量级外链资源聚合与导航系统。项目定位为“技术向外部参考资源的结构化索引工具”，旨在帮助开发者、研究者与运维人员快速定位高质量、细分类的互联网视频与媒体资源站点。LinkMatrix 不存储任何第三方内容，仅提供可校验的公开 URL 引用与分类视图，适用于个人书签管理、团队知识库外链整理、以及自动化爬虫的入口编排场景。

## 功能概览

- **分类导航视图**：按媒体类型、内容主题、更新频率等多维度对收录站点进行标签化分组，支持自定义分类标签。
- **裸链接校验模块**：对用户提交的 URL 进行可访问性探测与协议一致性检查，自动标记失效或重定向链接。
- **静态站点生成**：基于配置文件一键生成纯静态 HTML 导航页面，可直接部署至 Nginx、CDN 或对象存储服务。
- **外链元数据提取**：自动抓取目标站点的 Title、Description、H1 标题等基础元信息，辅助人工编辑决策。
- **变更追踪日志**：记录每次资源列表的增删改操作，支持回滚至任意历史版本，保障外链集合的可追溯性。
- **多格式导出**：支持 JSON、YAML、Markdown Table 三种格式导出，便于集成至 CI/CD 流水线或监控告警系统。
- **权限分级草案**：内置简单的只读/编辑双角色设计，适用于团队内部的外链维护协作场景。

## 应用场景

- **个人开发者知识库**：开发者可将日常积累的参考视频站、技术演讲录播平台统一收录，配合分类标签实现快速检索，避免重复搜索。
- **团队文档外链治理**：技术团队在撰写项目文档或运维手册时，频繁引用外部视频资源，LinkMatrix 可作为统一引用源，减少文档中的硬编码分散链接。
- **自动化爬虫入口管理**：数据采集工程师可将 LinkMatrix 导出的 JSON 作为爬虫启动种子列表，定期同步更新，降低入口维护成本。
- **社区资源分享页**：开源社区或技术论坛可使用 LinkMatrix 生成公开导航页，为成员提供高质量的外部媒体参考索引。
- **本地开发环境测试用数据源**：前端或测试工程师可将 LinkMatrix 作为 mock 外链数据源，模拟真实网络环境下的链接解析与渲染逻辑。

## 快速开始

以下步骤适用于 Linux / macOS / Windows WSL 环境。

```bash
# 克隆仓库
git clone https://github.com/your-org/linkmatrix.git
cd linkmatrix

# 安装依赖（使用 Python 3.10+）
pip install -r requirements.txt

# 初始化配置并运行本地服务
cp config.example.yaml config.yaml
python run.py --serve --port 8080
```

执行后，访问 `http://127.0.0.1:8080` 即可查看导航首页。默认加载 `data/links.yaml` 中的示例资源列表。

## 贡献指南

1. **分支管理**：从 `main` 分支新建 `feature/xxx` 或 `fix/xxx` 特性分支，禁止直接向 main 提交。提交信息请遵循 Conventional Commits 规范。
2. **本地验证**：运行 `pytest tests/` 确保所有测试通过；执行 `mypy src/` 进行类型检查；使用 `pre-commit run --all-files` 触发代码风格自动修正。
3. **链接增删流程**：编辑 `data/links.yaml` 后，务必运行 `python run.py validate` 校验 YAML 语法与 URL 协议合规性，校验通过后方可提交。
4. **文档同步**：若新增功能或修改配置字段，需同步更新 `docs/` 下对应文档，并在 PR 描述中标注文档变更点。
5. **PR 描述要求**：必须包含变更目的、测试结果截图或日志、影响范围评估。至少一名项目维护者审核通过后合并。

## 常见问题

**Q：LinkMatrix 是否存储或缓存第三方视频内容？**
A：否。LinkMatrix 仅保存 URL 字符串及可公开访问的 HTML 元信息（标题、描述）。不下载、不转存、不代理任何音视频文件或流数据。所有缓存仅用于加速元数据展示，且缓存策略遵守 robots.txt 约定。

**Q：如何批量导入现有的链接集合？**
A：支持 CSV 与 JSON Lines 格式的批量导入。将文件放置于 `data/import/` 目录下，执行 `python run.py import --format json --path data/import/source.json`。系统会自动去重并合并至 `links.yaml`。

**Q：本地服务启动后，页面显示链接不可访问，但浏览器可以正常打开？**
A：此情况通常由网络环境或服务器出口 IP 限制引起。可修改 `config.yaml` 中的 `timeout` 与 `user_agent` 字段，或通过 `--proxy` 参数指定代理地址。同时检查目标站点是否对 `requests` 默认的 User-Agent 进行了拦截。
