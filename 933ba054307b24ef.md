# ShuttleLinks

ShuttleLinks 是一个轻量级、社区驱动的技术资源导航与外部链接聚合系统。项目定位为开发者在信息过载环境下的高效筛选器，专注于收录高质量、垂直领域的工具站点与媒体资源，而非提供通用搜索引擎式服务。目标用户包括运维工程师、全栈开发者、技术决策者以及开源爱好者。ShuttleLinks 通过人工审核与自动化可用性检测相结合的方式，维护一份低冗余、高存活率的链接清单，帮助用户降低信息筛选成本，并围绕这些资源构建使用场景说明与组合推荐。

## 功能概览

- **智能链接分类与标签系统**：每个收录的 URL 均附带多级分类标签（如 流媒体工具、域名检测、网络诊断）及状态标记（可用性、响应时延、SSL 证书剩余天数），支持组合筛选。

- **自动化可用性巡检**：后端任务每天定时对全部外链执行 HTTP/HTTPS 可达性检测，并记录响应时间与状态码变化趋势，异常链接自动降级至待复审池。

- **社区投票与评论轻量化互动**：注册用户可对链接进行“有用/无用”投票，并提交简短的文字使用心得，优质评论会被置顶，辅助其他用户判断资源适用性。

- **个人收藏集与自定义分组**：用户可将常用链接添加至个人收藏夹，并创建自定义分组（例如“运维工具箱”、“视频源备用组”），支持分组内批量导出为 JSON 或 Markdown 列表。

- **RSS 订阅与变更通知**：对关注的分类或特定链接提供 RSS 输出，当链接状态变更（如 IP 变动、证书更新）或有新的推荐链接入库时，订阅端可收到更新提醒。

- **开放 API 与第三方集成能力**：提供 RESTful API 接口，允许外部工具查询链接状态、获取随机推荐列表或提交新链接审核，方便嵌入内部运维仪表盘或自动化脚本。

- **黑暗主题与响应式布局**：前端界面原生支持亮色/深色主题切换，并在移动端、平板、桌面设备上均保持良好的阅读与操作体验。

## 应用场景

- **运维故障排查时的备用工具快速定位**：当主用监控平台或常用检测网站出现访问异常时，运维人员可通过 ShuttleLinks 的“网络诊断”分类迅速找到备用的域名解析检测、端口扫描或 HTTP 状态检查工具，缩短故障定位时间。

- **内容聚合与媒体资源整理**：内容运营人员或视频创作者可使用 ShuttleLinks 管理多个视频源站或素材站链接，利用自定义分组按项目或主题归类，并随时导出分享给团队成员，避免浏览器收藏夹杂乱无章。

- **技术调研阶段的资源对比**：架构师或技术选型负责人在评估同类工具（如不同的轻量级 HTTP 服务、数据库管理面板）时，可通过 ShuttleLinks 的标签筛选快速列出候选站点，结合社区评论了解各工具的优缺点与实际使用体验。

- **开源项目文档中的外部依赖指引**：开源项目维护者可将 ShuttleLinks 中收录的稳定资源链接作为项目 README 或 Wiki 的“相关工具”推荐列表，减少维护大量外部链接的更新负担，并借助 ShuttleLinks 的可用性检测自动提示链接状态。

## 快速开始

以下步骤适用于在 Linux 或 macOS 开发环境本地部署 ShuttleLinks 服务。

```bash
# 克隆代码仓库
git clone https://github.com/shuttlelinks/shuttlelinks.git
cd shuttlelinks

# 安装项目依赖（使用 pip 与虚拟环境）
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# 初始化本地配置文件与 SQLite 数据库
cp .env.example .env
python manage.py migrate

# 启动开发服务器（默认监听 8000 端口）
python manage.py runserver
```

## 常见问题

**问：收录的外部链接如果失效了怎么办？**

系统后台每天定时执行可用性检测，若某个链接连续两次检测超时或返回 4xx/5xx 状态码，该链接会被自动标记为“异常”，并移出默认推荐列表。管理员会定期复审异常链接，若确认失效则移至归档区。用户也可以点击链接页面上的“报告失效”按钮主动反馈。

**问：我提交的新链接为什么没有立即显示？**

所有用户提交的链接均进入人工审核队列，审核主要检查内容合规性、分类准确性以及是否与已有链接重复。审核通常在 24 小时内完成，通过后链接状态初始为“待检测”，待首次巡检成功后方可正式对外可见。

**问：能否导出我收藏的全部链接，并在其他工具中使用？**

可以。在个人收藏夹页面，点击“导出”按钮，可选择 JSON 或 Markdown 表格格式。导出的数据包含链接标题、原始 URL、自定义标签和备注字段，便于迁移至浏览器书签管理或团队内部文档。
